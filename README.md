# LNEH Professional Website

GitHub/Vercel-ready static website inspired by the approved LNEH mockup.

## Included
- `index.html` homepage
- `shop.html` demo shop page
- `pages/` with About, Approach, Research, Ingredients, Contact
- `assets/images/` including the uploaded LNEH logo and visual assets from the approved mockup
- `assets/styles.css`
- `assets/script.js`
- `vercel.json`

## Upload to GitHub
1. Create a new GitHub repository.
2. Extract this ZIP.
3. Upload all files and folders inside `lneh_github_vercel_site` to the repository root.
4. Commit changes.

## Deploy on Vercel
1. Go to Vercel and click **Add New Project**.
2. Import the GitHub repository.
3. Keep framework as **Other** or static.
4. Click **Deploy**.

## Important
This is a static demo shop. Buttons show demo cart alerts only. Connect to Shopify, WooCommerce, Snipcart, Stripe, or a custom backend later.
